CCNode节点

2 minutes read

属性

属性名 描述 范例
children 获得指定节点所有子节点的数组
active 打开/关闭节点 this.node.active = false
parent 更改节点的父节点 this.node.parent = parentNode等价于代码: this.node.removeFromParent(false)parentNode.addChild(this.node)
x/y 更改节点位置 this.node.x = 100
rotation 更改节点旋转 this.node.rotation = 90
scaleX/scaleY 更改节点缩放 this.node.scaleX = 2
width/height 更改节点尺寸 this.node.width = 100
color 更改节点颜色 this.node.color = cc.Color.RED
opacity 更改节点不透明度 0 ~ 255 this.node.opacity = 128
enable 组件开关, 是否执行update方法, 是否渲染(draw方法)

方法

方法名 描述 范例
getComponent() 获取节点上的组件 this.node.getComponent(cc.Label).stringthis.nodeA.getComponent(“ScriptA”).func()
getChildByName() 获得指定名字子节点 this.node.getChildrenByName(“Child1”)
setPosition(x, y) 设置节点位置 this.node.setPosition(100, 50) | this.node.setPosition(cc.v2(100, 50))
setRotation(r) 设置节点旋转 this.node.setRotation(90)
setScale(scaleX, scaleY) 设置节点缩放 this.node.setScale(2, 2) | this.node.setScale(2)
setContentSize(w, h) 设置宽高 this.node.setContentSize(100, 100) | this.node.setContentSize(cc.v2(100, 100))
addComponent() 为节点添加组件 this.addComponent(cc.Sprite)
destroy() 销毁节点 见下文

查找节点

根据路径查找节点: cc.find("Canvas/Menu/Back") 

cc.Class({
	extends: cc.Component,
	
	properties: {
		target: cc.Node,
	},
	
    onLoad(){
        this.target = cc.find('Canvas/nodeB')
    },
})

创建新节点

用脚本动态创建节点: new cc.Node

cc.Class({
	extends: cc.Component,
	
	properties: {
		sprite: cc.SpriteFrame,
	},
	
	start: function(){
		var node = new cc.Node("NewSprite");    // 节点名字
		var sp = node.addComponent(cc.Sprite);
		sp.spriteFrame = this.sprite;
		node.parent = this.node    // 指定父节点, 没有父节点的节点不会渲染, 并会在此帧结束后自动释放
	}
})

复制已有节点

复制场景中已有节点: cc.instantiate

cc.Class({
	extends: cc.Component,
	
	properties: {
		target: cc.Node,
	},
	
	start: function(){
		var node = cc.instantiate(this.target);
		node.parent = this.node;
		node.setPosition(0, 0)
	}
})

创建预制节点

cc建议开发者更多地使用创建与直接点方法动态创建节点

cc.Class({
	extends: cc.Component,
	
	properties: {
		target: {
			default: null,
			type: cc.Prefab,
		}
	},
	
	start () {
		var scene = cc.director.getScene();
		var node = cc.instantiate(this.target)    // 将预制作为第一个参数传入, 然后设置父节点与坐标
		node.parent = scene;
		node.setPosition(0, 0)
	}
})

销毁节点

销毁节点不会立刻从内存中移除, 而是在当前帧逻辑更新结束后统一执行, 当一个节点销毁后, 该节点就处于无效状态, 为防止某些未及时停止的异步回调, 可以通过cc.isValid判断当前节点是否已经被销毁.

cc.Class({
	extends: cc.Component,
	
	properties: {
		target: cc.Node,
	},
	
	start () {
		setTimeout(function () {
			this.target.destroy();
		}.bind(this), 5000)
	},
	update(dt){
		if (cc.isValid(this.target)){    // 安全写法, 确保this.target的有效性
			this.target.rotation += dt * 10.0
		}
	}
})
