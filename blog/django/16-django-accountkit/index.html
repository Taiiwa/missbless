Django Accountkit

June 12, 2020 Issac loveless

1 minute read

facebook开放平台

此软件包允许使用带有django身份验证的facebook accountkit

如果您想了解更多有关Facebook AccountKit的信息，请查看以下资源

  • https://auth0.com/blog/facebook-account-kit-passwordless-authentication/
  • https://developers.facebook.com/docs/accountkit

要求

Python（>；=2.7，<；=3.7） django（>；=1.11）

安装

使用pip

pip install django-accountkit

然后要将django accountkit添加到项目中，请将应用程序accountkitlogin添加到INSTALLED_APPS。

现在在settings.py文件中添加以下设置

APP_ID = <Accountkit App ID>
ACCOUNT_KIT_APP_SECRET = <Accountkit App Secret>
ACCOUNT_KIT_VERSION = "v1.0"
ACCOUNT_KIT_SUCCESS_REDIRECT = <The URL for the page where user will land after authenticating> (Please Use absolute path only)
#Example ACCOUNT_KIT_SUCCESS_REDIRECT = "http://localhost:8000/success"

然后在settings.py文件中添加以下内容

AUTHENTICATION_BACKENDS = (
    'accountkitlogin.authenticate.GetOrCreateUser',
)

使用accountkitlogin

视图

使用accountkit验证用户导入from accountkitlogin.views import login_status

然后在您的视图中使用login_status

@csrf_exempt
def success_page(request):
    context = login_status(request)

注意，csrf豁免是必需的，因为facebook accountkit会将用户重定向到您的成功页面url，这可能会引发跨源错误

login_status函数接受请求作为参数，并返回一个字典，其中

  • 密钥“authenticated”为真或假
  • 键“消息”与用户是否经过身份验证有关的消息
  • 当“authenticated”为true时，键“user”返回用户实例

为您在ACCOUNT_KIT_SUCCESS_REDIRECT设置中定义的成功页面url添加url.py文件的视图

模板标签

在模板开始处添加{% load accountkit %}，并将以下模板标记添加到要在其中显示登录按钮的文件中

  1. 在标签中添加{% accountkitjs %}
  2. 在标签中添加{% accountkitform %}
  3. 在body部分的任何地方添加{% smslogin %}和/或{% emaillogin %}以显示登录按钮。这两个标记都接受两个参数classname和text，您可以在其中添加自定义css类和要在按钮上显示的文本。示例{% emaillogin "button" "Use the email" %}
