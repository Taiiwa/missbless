Element上传头像到七牛云

June 2, 2020 Issac loveless

2 minutes read

 Twitter

 Facebook

 Reddit

 LinkedIn

 Pinterest

 Email

逻辑：

通过post请求七牛云的上传接口，可以将图片上传到七牛云。 post除了携带文件以外，还需要传递令牌。令牌通过七牛云验证AK和SK来返回给开发者。 还有一个可选参数key，作为上传文件的文件名。如果上传key，则用该key，如果没有上传，则由七牛云生成一个文件id。 key不可重复，可以生成一个uuid作为key。 element在template的标签中发起请求，我们无法接受post请求返回的文件名拼接url，所以事先写好key，如果上传成功，则将此key与七牛云的url进行拼接，从而实现图片预览的效果。

前端element

<template>
    <el-upload
        class="avatar-uploader"
        :auto-upload="true"
        :data='qiniuData'
        action="http://up-z1.qiniu.com"
        :show-file-list="false"
        :on-success="handleAvatarSuccess"
        :before-upload="beforeAvatarUpload">
        <img v-if="imageUrl" :src="imageUrl" class="avatar">
        <i v-else class="el-icon-plus avatar-uploader-icon"></i>
    </el-upload>
</template>

<script>
import { uptoken_get } from "../axios_api/api";
export default {
    data() {
        return {
            imageUrl: "",
            qiniuData: {
                //上传图片携带的参数
                token: "",
                key: ""
            },
        }
    },
mounted() {
    this.get_uptoken();
},
methods:{
    // 生成uuid作为图片key
    guid2() {
    function S4() {
        return (((1 + Math.random()) * 0x10000) | 0).toString(16).substring(1);
    }
    return (
        S4() + S4() +"-" +S4() +"-" +S4() + "-" + S4() + "-" + S4() + S4() + S4()
    );
    },

    // 获取七牛云凭证
    get_uptoken() {
    uptoken_get().then(res => {
        console.log("****************");
        this.qiniuData.token = res.up_token;
        console.log(this.qiniuData);
    });
    // 生成key
    this.qiniuData.key = this.guid2();
    },

    // 上传头图
    post_img() {
    return "https://jsonplaceholder.typicode.com/posts/";
    },

    beforeAvatarUpload(file) {
    const isJPG = file.type === "image/jpeg";
    const isPNG = file.type === "image/png";
    const isLt2M = file.size / 1024 / 1024 < 2;

    if (!isJPG && !isPNG) {
        this.$message.error("上传头像图片只能是 JPG/PNG 格式!");
        return isJPG && isLt2M;
    }
    if (!isLt2M) {
        this.$message.error("上传头像图片大小不能超过 2MB!");
        return isJPG && isLt2M;
    }
    },

    // 上传成功拼接图片url
    handleAvatarSuccess(res, file) {
    this.imageUrl = "http://qakgy27pn.bkt.clouddn.com/" + this.qiniuData.key;
    this.ruleForm.img = this.qiniuData.key;
    },

}

后端七牛云凭证接口

from rest_framework.views import APIView
from rest_framework.response import Response
from admin_keys.config import AK, SK
from qiniu import Auth

class QiNiu(APIView):
    def get(self, request):
        # 声明密钥对象
        q = Auth(AK, SK)
        # 生成令牌
        token = q.upload_token('missty520')
        return Response({'up_token': token})
